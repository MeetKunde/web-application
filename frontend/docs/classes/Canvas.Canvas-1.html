Class Canvas

Static class for representation of Konva.Stage - HTML5 Canvas. It extends Konva library objects by functions to easy draw basic geometry shapes, to resolve clicks on created shapes, to prompt during creating new shapes and to make created shapes interactive. Class contains many constant variables with shapes properties which are used in creating all shapes.
